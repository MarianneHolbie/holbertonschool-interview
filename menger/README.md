# Menger Sponge

This project implements a function to draw a 2D Menger Sponge in C.

![Menger Sponge](https://paulbourke.net/fractals/polyhedral/carpet.gif)

## Description

A Menger Sponge is a fractal that generalizes the Cantor set and the Sierpinski carpet to three dimensions. Here, we draw a 2D version of the Menger sponge, but the principle is the same.

- A level N sponge is a 3x3 square of level N-1 sponges, except for the center one, which is left empty.
- A level 0 sponge is represented by the `#` character.

## Function Prototype

```c
void menger(int level);
```

## Compilation
To compile the program, you can use the following command in your terminal:

```bash
gcc -Wall -Wextra -Werror -pedantic -o 0-menger 0-menger.c 0-main.c -lm
```
